# 화면 사양 & 디자인 토큰

## 디자인 토큰
레퍼런스 2장에서 추출: ①월간 캘린더에 날짜별 대표 이미지가 박힌 시각적 타임라인 ②사진 위 메모가 타이핑된 포스터형 일기. 원칙은 **미니멀 타이포 + 여백 중심**.

- 배경: `#FAFAF7` (따뜻한 오프화이트) / 카드 표면 `#FFFFFF`
- 잉크: `#1C1C1E` / 보조 텍스트 `#8A8A8E` / 분리선 `#ECEAE3`
- 빈 날 점: `#D9D6CC`
- 액센트: `#C8745A` (테라코타, 절제된 포인트)
- 감정 색: joy `#F6C453` · calm `#9DB6D8` · neutral `#C8C6BD` · sad `#8FA3B0` · tired `#B7A6C9` · love `#E79CB0`
- 폰트: 본문 system-ui, 카드/날짜 숫자는 serif (`Georgia, 'Noto Serif KR'`)로 일기 질감
- 라운드: 카드 14px / 셀 10px / 칩 999px
- 모바일 프레임: 데스크톱에선 max-width 430px 폰 프레임 중앙 배치

## 1. 월간 캘린더 홈 (MonthCalendarScreen) — 첫 화면
- 상단: ‹ **2026년 6월** › 네비 + 스트릭(🔥 n일) + 잠금 아이콘
- 요일 헤더(일~토), 일요일/토요일 색 구분
- 7열 그리드. 각 셀:
  - 사진 있는 날 → 대표 썸네일이 셀을 채우고 좌상단 날짜 숫자(가독 위해 그라데이션)
  - 엔트리 있고 사진 없는 날 → 감정색 옅은 틴트 + 날짜
  - 빈 날 → 미묘한 점 하나 (아직 기록 안 함을 느끼게)
  - 오늘 → 링 강조
- 셀 탭 → 해당 날짜 에디터. 하단 + (오늘 기록) FAB.
- 월 요약 바: 기록 일수 / 대표사진 비율, "월간 리캡 보기" 진입.

## 2. 하루 기록 작성 (DayEntryScreen) — "하루 카드 편집기"
입력 순서는 **사진 먼저, 문장은 다음**.
- 날짜 헤더 + 닫기/저장
- 사진 영역: 대표 1 + 보조 ≤3 (총 4). 추가/삭제/대표 지정. import 시 다운스케일.
- 오늘 한 줄 (단문 input)
- 짧은 회고 (작은 textarea)
- 감정 선택 (칩)
- 태그 (엔터로 추가되는 칩)
- 메모(선택, 접힘)
- 저장 → 캘린더 셀 썸네일 즉시 반영 후 홈 복귀. 삭제 액션.
- Acceptance: 사진 0장이어도 저장 가능(감정/한 줄만으로). 대표 미지정 시 첫 사진 자동 대표.

## 3. 하루 카드 출력 (DayCardScreen)
- 저장된 하루를 공유용 카드로 렌더. 템플릿 3종 탭 전환:
  - **포스터형**: 대표사진 풀블리드 + 하단 그라데이션 위 한 줄/날짜 타이핑
  - **종이 일기장형**: 종이 질감 배경, 사진 폴라로이드, 손글씨풍 본문
  - **미니멀형**: 사진 중심 + 여백, 날짜·한 줄 소형
- 캔버스 1080×1350(4:5) 렌더 → **PNG 다운로드**. 배경화면/공유용.
- Acceptance: 사진 없는 엔트리도 텍스트 카드로 렌더된다.

## 4. (덤) 월간 리캡 · On This Day
- 월간 리캡: 그 달 대표 asset만 모아 한 줄 캡션+날짜 슬라이드 자동재생.
- On This Day: 과거 같은 날짜(작년 오늘 등) 대표사진을 홈 상단 카드로 노출.

## 네비게이션
`#/calendar` (기본) ↔ `#/entry/YYYY-MM-DD` ↔ `#/card/YYYY-MM-DD` ↔ `#/recap/YYYY-MM`. 해시 라우팅.
